Iron Duck !

Hello and welcome to our Iron Duck game page !

Developped by Marion & Hicham

We have developped this small game as a part of our first project with Ironhack. 

It was a great challenge, and a cool way to learn new things. 

This game was created using HTML, CSS and JS

The instructions are quite simple, avoid incoming obstacles by moving with Z & D (or left and right arrow) and jumping with space. You get points by succesfully avoiding obstacles. And if you get hit, well it's Game Over and you'll have to try it again.

There are 3 diffrent universes, each with its own Iron Duck ! Space Duck, Beach Duck and City Duck. Avoid all hazards as long as possible !
